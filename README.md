# mnist_digit

Each folder contains python notebooks to create a model for mnist dataset. Submission.cvs are the outputs for each notebook.

The accuracies of different models on mnist dataset is given below.

Data and machine provided by kaggle are used to generate the results.

| Model used              | Accuracy |
|-------------------------|----------|
| Decision trees          | 76.47    |
| k nearest neighbours    | 92.34    |
| kernel svm              | 88.47    |
| logistic regression     | 84.25    |
| naive bayes             | 56.25    |
| random forest           | 94.98    |
| support vector machines | 84.257   |
